✅ Rangkuman Tugas Tes Fullstack Developer
1. Backend (Golang + go-chi v5)

Kamu harus membuat RESTful API sistem manajemen buku dengan endpoint berikut:

Endpoint Wajib

GET /books
Mengembalikan daftar semua buku.

GET /books/{id}
Mengambil detail buku berdasarkan ID.

POST /books
Menambah buku baru.
Contoh body JSON:

{
  "title": "Book Title",
  "author": "Author Name",
  "published_year": 2023
}


PUT /books/{id}
Memperbarui informasi buku.

DELETE /books/{id}
Menghapus buku berdasarkan ID.

Persyaratan Teknis Backend

Gunakan go-chi versi v5 untuk routing.

Data hanya disimpan di memory (slice/map).

Response harus JSON.

Struktur kode harus modular (pisahkan handler, model, router).

Buat custom middleware LoggerMiddleware untuk mencatat:

method,

path,

response time.

Tambahkan komentar pada setiap fungsi.

Gunakan Singleton Pattern jika ingin bonus (untuk instance storage).

Terapkan prinsip DRY (untuk error handling & response formatting).

2. Frontend (ReactJS / VueJS / AngularJS)

Kamu harus membuat halaman antarmuka yang berinteraksi dengan API backend di atas.

Kewajiban Frontend

Menampilkan daftar buku.

Form untuk tambah buku.

Halaman/form untuk edit buku.

Tombol delete.

Tampilan harus user-friendly.

Sertakan alasan kenapa desain yang dibuat mudah digunakan user.

Bonus Frontend (Opsional)

Gunakan komponen penting dari framework dan jelaskan kenapa penting.
